# Development Setup

If you need to set up a GNS3 development environment or manually install the GNS3 server, please refer to the GNS3 Server development setup guide:

👉 [GNS3 Server Development Setup Guide](https://github.com/GNS3/gns3-server/blob/3.1/docs/development-setup.md)

This guide is based on Ubuntu 24.04 and covers:

- System dependency installation and Docker configuration
- Source code cloning and Python virtual environment setup
- AI Copilot optional dependencies
- Starting the development server

You can also download the GNS3 VM to use a pre-configured environment without manual setup.
