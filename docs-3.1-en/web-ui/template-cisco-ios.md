# How to create a Cisco IOS template

:::tip Video Tutorial

📺 **[Watch the Video Tutorial](https://www.youtube.com/watch?v=59k-f5qhHO0)** - Follow along step-by-step to create a Cisco IOS template

:::

This guide uses the Cisco IOS 7200 as an example to show how to create an IOS template.

1. In the new template wizard, click the "Next" button.
   <img style={{ width: '100%' }} alt="Click Next" src={useBaseUrl('img/web-ui/zh/template-click-next.jpeg')} />

2. Type "7200" in the search box to search quickly. Then click the button marked "2" in the image.
   <img style={{ width: '100%' }} alt="Search 7200" src={useBaseUrl('img/web-ui/zh/template-search-7200.jpeg')} />

3. Use `c7200-adventerprisek9-mz.152-4M11.image` as an example. Follow the order shown in the image. Select and upload your local image file.
   <img style={{ width: '100%' }} alt="Upload image file" src={useBaseUrl('img/web-ui/zh/template-upload-image.jpeg')} />

4. After the upload finishes, the status of `c7200-adventerprisek9-mz.152-4.M11.image` changes from `Missing` to `Ready`. Click the "Create" button as shown in the image.
   <img style={{ width: '100%' }} alt="Image ready, click Create" src={useBaseUrl('img/web-ui/zh/template-ready-create.jpeg')} />

5. Enter a template name. The default name is `Cisco 7200`. Click the "Add Template" button to finish.
   <img style={{ width: '100%' }} alt="Fill in the template name" src={useBaseUrl('img/web-ui/zh/template-name-finish.jpeg')} />
