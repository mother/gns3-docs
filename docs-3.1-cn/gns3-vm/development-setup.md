# 开发环境搭建

如果您需要搭建 GNS3 开发环境或手动安装 GNS3 服务器，请参考 GNS3 Server 项目的开发环境搭建指南：

👉 [GNS3 Server Development Setup Guide](https://github.com/GNS3/gns3-server/blob/3.1/docs/development-setup.md)

该指南基于 Ubuntu 24.04，涵盖以下内容：

- 系统依赖安装与 Docker 配置
- 源码克隆与 Python 虚拟环境搭建
- AI Copilot 依赖安装（可选）
- 开发服务器启动

您也可以下载 GNS3 VM 直接使用预配置的环境，无需手动搭建。
