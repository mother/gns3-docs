如何创建 Cisco IOS 模板

视频教程

📺 观看视频教程 - 跟随视频一步步创建 Cisco IOS 模板

以 Cisco IOS 7200 为例，介绍如何创建 IOS 模板。

  1. 在新建模板向导中点击"Next"按钮。

  2. 在搜索框输入"7200"快速搜索，然后点击图中"2"标识的按钮。

  3. 以 c7200-adventerprisek9-mz.152-4M11.image 为例，按照图中所示顺序点击，选择并上传本地的镜像文件。

  4. 上传完成后，c7200-adventerprisek9-mz.152-4.M11.image 状态从 Missing 变为 Ready，按下图所示点击 "Create" 按钮。

  5. 输入模板名称，这里默认为 Cisco 7200，点击 "Add Template" 按钮完成创建。
