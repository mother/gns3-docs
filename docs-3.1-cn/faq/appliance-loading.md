# 设备模板加载

### GNS3 从哪里加载设备模板（.gns3a）文件？优先级是怎样的？

GNS3 从两个目录加载 `.gns3a` 设备模板文件：

1. **内置设备目录**：`~/.local/share/GNS3/appliances/`
   - 存放从 GNS3 registry 自动下载的设备
   - 由系统自动维护更新

2. **自定义设备目录**：`~/GNS3/appliances/`
   - 存放用户自定义或修改过的设备模板文件
   - 由用户手动管理

**加载优先级：** 系统先加载内置设备目录，再加载自定义目录。如果两个目录中有相同 `device_id` 的设备，自定义目录中的文件会覆盖内置文件。这种设计允许用户自定义设备而不会被 registry 的自动更新覆盖修改。
