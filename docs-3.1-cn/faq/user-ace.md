# 用户管理与 ACE 规则

### 删除用户时，已配置的 ACE 权限规则会怎样？

如果用户被 ACE（Access Control Entry）规则适用，删除用户时会自动执行以下清理：

1. **自动删除该用户自己的 ACE** — 删除 `ace_type='user'` 归属该用户的所有规则
2. **删除路径引用** — 删除所有指向 `/users/{user_id}` 路径的 ACE（包括组权限中对该用户的引用）
3. **无孤立引用** — 删除后不会留下孤立的 ACE 引用记录
