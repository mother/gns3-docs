# GNS3 3.10 文档

欢迎阅读 GNS3 3.10 版本文档。本文档介绍 GNS3 3.10 的主要功能和使用方法。

## 主要新特性

- **全新的 Web UI** - 支持Web控制台、Web VNC和Web Wireshark
- **Web UI 跨代升级** - 前端框架从 Angular 14 升级到 Angular 21，得益于 AI 辅助完成代码迁移
- **AI 功能集成** - 故障注入和数据包分析
- **GNS3 VM** - 改进的虚拟机性能和易用性
- **多用户协作** - 支持团队协作和权限管理

## 快速开始

从左侧导航栏选择您感兴趣的主题开始学习：

- **GNS3 VM** - 了解如何在不同虚拟化平台中运行 GNS3 VM
- **虚拟化平台指南** - VirtualBox、KVM、Hyper-V、VMware 等平台的使用说明
